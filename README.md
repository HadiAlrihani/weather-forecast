# Weather Forecast App

A web weather application that allows users to search for cities worldwide and view current and 5-day weather forecasts.

**Live Demo:** https://weather-app-ten-ruby-17.vercel.app/

---

## Features

- Search weather by city
- Current weather conditions (Temperature, feels like, wind speed)
- 5-day highs and lows
- Location autocomplete for accurate searches
- Responsive design
- Dark mode support

---

## Tech Stack

- **Frontend:** React, Vite
- **Styling:** Tailwind CSS
- **APIs:** OpenWeatherMap, LocationIQ
- **Deployment:** Vercel
